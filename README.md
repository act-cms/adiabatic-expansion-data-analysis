Determining Heat Capacity Ratios via Experimental Data and Statistical Mechanics
===================

Students will examine experimental and computational data to compute heat capacity ratios.
This module combines analysis of time-resolved pressure measurements with electronic structure calculations to connect experimentally measured thermodynamic behavior to molecular-level statistical thermodynamics.
This module is designed for an upper-level undergraduate Physical Chemistry lab.
